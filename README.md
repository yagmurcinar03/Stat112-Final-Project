# Laptop Market Analysis & Data Cleaning Project

## Overview
This project analyzes laptop market data to understand pricing trends, product features, and user preferences.

## What We Did
- Cleaned and prepared raw dataset using Python (Pandas, NumPy)
- Handled missing values, duplicates, and inconsistent formats
- Conducted exploratory data analysis on price, brand, processor, and other features
- Created visualizations to identify patterns and trends

## Key Insights
- Laptop prices increased significantly between 2020–2023
- Processor type has a strong impact on price distribution
- Lightweight laptops are more preferred by users
- Different brands show different pricing behaviors depending on condition

## Tools
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)

- cleaning.py → data cleaning process
- visualization(1).py → data visualization and analysis
